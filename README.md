Kiosk Mode Sample
===================================

This sample shows how the KioskMode class can allow your device to enter Kiosk Mode.

Introduction
------------

This sample app shows how to develop code to enter your device into Kiosk Mode. 

Kiosk Mode locks your app down
to either a couple of selected apps or a single app. Additionally, the user can restrict certain features such
as settings changes, expansion of the status bar, certain hardware keys and more.

Pre-requisites
--------------

Knox SDK, device that supports Knox API level 24
